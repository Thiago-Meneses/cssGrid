CSS Grid

In this post we are going learn about the CSS Grid Layout Module â€“ also called CSS Grid

The CSS Grid:

  • Is a Layout System, evidently one that is grid-based
  • It has rows and columns
  • Makes it easir to design pages without the use of floats and positioning

Grid Elements

The Grid is made up of 2 elements:

  • The Parent Element
  • The Child Elements , made up of one or multiple elements

Grid Columns

Columns represent the Vertical Lines of the grid items

Grid Rows

Rows represent the Horizontal Lines of the grid items

Grid Properties

The Essential Properties

display

This property is extremely important

It is responsible for turning an HTML element into a Grid Container

When dealing with CSS Grid, we can use either:

  • display: grid or
  • display: inline-grid

All direct children of the grid container automatically become Grid Items.

grid-template-columns

This property is used to define:

  • The number of columns
  • The width of each column

1. Example

Create a grid layout with 4 columns, each being 100px wide


        grid-template-columns: 100px 100px 100px 100px;

2. Example

Make a grid layout containing 3 columns, where 2 columns are 100px wide and one is 200px wide


        grid-template-columns: 100px 100px 200px;

3. Example

Make a grid layout containing 4 columns, all with the same width


        grid-template-columns: auto auto auto auto;

*The value auto makes all columns have the same width;
* If the number of items exceeds the number of columns, the grid will automatically add a new row to put the items in.

grid-template-rows

This property is used to:

  • Define the Height of each row

Example:

Make a grid layout with 2 rows, each one having a with a height of 100px

                                                        
        grid-template-rows: 100px 100px;

*If the number of items is greater than the amount of rows, a new row will be created, even if it was not defined in the property

The Styling Properties

justify-content

This property is used to horizontally align all the grid items inside of the container

To do this, it needs to alter the distance between the columns of the grid

Basically, the justify-content property:

  • Handles the horizontal leftover space that exists inside the container
  • Makes the grid items occupy the space in some specific way that we wan

The premise is: there needs to be horizontal leftover space inside the container

To fulfill this premise, the total width of grid items must be smaller than the container itself, otherwise there is no free space availabe and the property won't work

Values Accepted:

  • justify-content: space-evenly â†’ will give the columns equal amount of space between, and around them
  • justify-content: space-around â†’ will give the columns equal amount of space around them
  • justify-content: space-between â†’ will give the columns equal amount of space between them
  • justify-content: center â†’ will align the grid in the middle of the container
  • justify-content: start â†’ will align the grid at the beginning of the container
  • justify-content: end â†’ will align the grid at the end of the container

align-content

This property is used to Vertically align all the grid items inside of the container

To do this, it needs to alter the distance between the rows of the grid

Basically, the align-content property:

  • Handles the vertical leftover space that exists inside the container
  • Makes the grid items occupy the space in some specific way that we want

The premise is: there needs to be vertical leftover space inside the container

To fulfill this premise, the total height of grid items must be smaller than the container itself, otherwise there is no free space availabe and the property won't work

Values Accepted:

  • align-content: space-evenly â†’ will give the rows equal amount of space between, and around them
  • align-content: space-around â†’ will give the rows equal amount of space around them
  • align-content: space-between â†’ will give the rows equal amount of space between them
  • align-content: center â†’ will align the rows at the beginning of the container
  • align-content: start â†’ will align the grid at the beginning of the container
  • align-content: end â†’ will align the rows at the end of the container

Grid Gaps

Gaps are the Spaces between each column and between each row

We can change the size of our Grid Gaps

We have special properties just for this:


grid-column-gap

Responsible for changing the gaps between columns

grid-row-gap

Responsible for changing the gaps between rows

grid-gap

Shorthand property for both columns and rows

Grid Lines

Grid Lines are extremely important

They are located between Grid Items and are used as a reference

By default, Grid Items will only fill one line and one row

With the help of Grid Lines we can define them to span multiple columns and rows

In short, Grid Lines help us to define the amount of columns and rows that a certain Grid Item will occupy.

There are 2 types of Grid Lines:

Column Lines

Theses lines are the ones between the columns of the grid.

Row Lines

Theses lines are the ones between the rows of the grid.

Grid Lines Properties

There are 7 properties directly correlated with Grid Lines:

grid-column-start

Determines the column from where the Grid Item will start

                                                    
Ex: grid-column-start: 1;
grid-column-end

Determines the column where the Grid Item will end

                                                    
Ex: grid-column-end: 5;
grid-column

Shorthand property, defines at which column the item will start in which column it will end.

                                                    
Ex: grid-column: 1 / 5;   	/* Will span from line 1 to line 5 */
Ex: grid-column: 1 / span 3; 	/* Will span 3 lines starting from line 1*/
grid-row-start

Determines the row from where the Grid Item will start

grid-row-end

Determines the row where the Grid Item will end

grid-row

Shorthand property, defines at which row the item will start and where it will end.

                                                    
Ex: grid-row: 1 / 5;   	/* Will span from line 1 to line 4 */
Ex: grid-row: 1 / span 3; 	/* Will span 2 lines starting from line 1*/
grid-area

Shorthand property for all of them

                                                    
Ex: grid-area: 1 / 2 / 5 / 6;   		/* Will go from row-line 1 to row-line 5 and from column-line 2 to column line 6:
Ex: grid-area: 2 / 1 / span 2 / span 3; 	/* Will span 2 rows from row-line 2 and span 3 columns from column-line 1;

Flexbox vs Grid

Flexbox is just a tool that was created to help us to layout items flexibily

To do this, Flexbox uses one dimension, horizontally.

This way the items will be flexible and able to change their size dinamically, based on the content inside of the flexbox container


Grid is exactly the same, in the sense that it allows us to layout items flexibly inside of the container

The difference lies in the fact that the Grid allows us to work vertically and horizontally, using 2 dimensions.

Grid by itself doesn't define any rows or columns

By default things will be looking exactly the same


Flexbox and grid were designed so they would work amazingly together, and using flexbox items inside of your grid containers is one of the best ways to lay out a site

Making a Webpage template

One of the main goals of the CSS Grid is allow us to create webpage templates

Learning how to create templates will allow you to make much better looking and better organized webpages.

Properties for your template

Before creating our templates, we will need to learn some new properties:

grid-area

We have already covered this property but now we will explain another one of its uses

It can also be used to assign names to Grid Items

This fact is extremely important, as the layout of our grid will behave accordingly with the names our Grid Items have received

Example:

Assign the name myArea to elements with class item1

                                                    
<styles>
.item1 {
    grid-area: myArea;
}
</styles>
<body>
<div>
    <div class="item1"> 1 </div>
    <div class="item2"> 2 </div>
    <div class="item3"> 3 </div>
</div>
</body>

grid-template-areas

This property is similar to the last one and this is because both work hand in hand.

It allows us to use the Grid Items that we previously named using the grid-area property

Now, thse named Grid Items will be used to help in designing our Grid Template


To better understand this property, we have to keep in mind the fact that the Grid Layout has 2 dimensions:

  • Horizontal Dimension - Represented by Rows
  • Vertical Dimension - Represented by Columns

With this property we will define each row and each column of our grid

We will be filling these 2 dimensions inside of the grid-template-areas property

To help us achieve our goal, this property has a simple sintax we will learn.

Example

Step 1:

Create a parent div, with a class grid container, containing 5 child divs, which will be the Grid Items

                                                    
<!-- HTML -->

<div class="grid-container">
<div class="item1">Header</div>
<div class="item2">Menu</div>
<div class="item3">Main</div>  
<div class="item4">Right</div>
<div class="item5">Footer</div>
</div>

Step 2:

Give a unique value to each one of our Grid Items using the grid-area property

                                                    
    /* CSS */
    
    .item1 {
    grid-area: header;
    }
    .item2 {
    grid-area: menu;
    }
    .item3 {
    grid-area: main;
    }
    .item4 {
    grid-area: right;
    }
    .item5 {
    grid-area: footer;
    }

Step 3:

Now we will attribute the necessary properties to our Container Class

                                                    
    /* CSS */
    
    .grid-container {
    background-color: #2196F3;
    padding: 10px;
    display: grid;
    grid-gap: 10px;
    grid-template-areas: /* This one we will be learn step by step */
    }

Learning to Use grid-template-areas

This property has a sintax we need to learn:

  1. The sintax starts in the first Row, from left to right
  2. Rows are be represented by apostrophes ''
  3. Here is where the Grid Items we named will be useful
  4. We will fill the interior of the apostrophes with the names we have given to each Grid Item

  5. Columns will be defined inside the apostrophes
  6. The amount of Grid Items we write will be equal to the total amount of columns in our layout
  7. The first name will represent the first column, the second name will be the second column, and so on...
  8. All of this happens inside '', meaning that we have to do one row at a time
